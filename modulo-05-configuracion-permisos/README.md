# Modulo 05: Configuracion y Permisos

## Descripcion

Sistema de configuracion multinivel de Claude Code y su sistema de permisos.
Aprenderas a controlar que puede hacer Claude Code, compartir configuraciones
con tu equipo y asegurar entornos.

## Objetivos

1. Comprender la jerarquia de 5 niveles de configuracion
2. Configurar permisos allow/ask/deny por herramienta
3. Dominar los modos de operacion
4. Activar y configurar sandbox
5. Disenar configuraciones por rol para equipos
6. Activar y gestionar Auto Mode con su sistema de seguridad dual

## Duracion: 2h 05min (80 min teoria + 45 min ejercicios)

## Prerequisitos: Modulos 01-04 completados
